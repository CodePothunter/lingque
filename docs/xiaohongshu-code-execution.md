# 我让 AI 帮我查磁盘空间，它直接在服务器上跑了 df -h

不是给我一段命令让我自己复制粘贴。

**它直接跑了，然后把结果告诉我。**

---

## 什么意思？

灵雀新增了两个工具：

- `run_bash`：直接在服务器上执行 shell 命令
- `run_claude_code`：启动一个 Claude Code 子进程处理复杂任务

所以当你说"帮我看下服务器还剩多少空间"——AI 不再说"你可以运行 df -h"。它自己跑 `df -h`，读结果，然后告诉你"还剩 42G，占了 73%"。

---

## 不怕它乱来吗？

怕。所以有安全黑名单：

```
rm -rf /        ❌
mkfs             ❌
shutdown         ❌
fork bomb        ❌
chmod -R 777 /   ❌
```

这是一个 `frozenset`，硬编码在代码里。不是 prompt 里写"请不要执行危险命令"——是代码层面直接拦截。

加了 `sudo` 前缀也一样拦。

---

## 输出不会撑爆

命令输出超过 10,000 字符自动截断。

你让它 `find / -name "*.log"` 也不怕——输出太长就切掉，附一句"输出已截断"。AI 知道结果不完整，会自己调整策略。

Bash 命令超时 60 秒。Claude Code 超时 5 分钟。跑飞了自动杀进程。

---

## 最厉害的：20 轮连续操作

AI 可以连续调用工具 20 轮。什么意思？

它可以：写代码 → 跑 → 看报错 → 改 → 再跑 → 成功

六步操作一气呵成，你只需要说一句"帮我写个脚本统计日志里的错误数"。
