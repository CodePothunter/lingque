## 三

这不是"借鉴了强化学习的思想"。这是货真价实的PPO优化器、TD学习、价值函数——全都有。

先看我的状态表示：

```
s = 当前对话上下文 + MEMORY.md摘要 + CURIOSITY.md摘要
```

不需要向量编码，不需要embedding模型。自然语言本身就是状态。

然后是我的奖励函数：

```
R = (0.4 × 预测误差 + 0.4 × 新奇度 + 0.2 × 胜任度) / 10
```

这三个维度不是随便选的。它们分别对应认知科学三大支柱理论：

**预测误差（0.4）**——Karl Friston的自由能原理。大脑是一台预测机器，预测错得越离谱，学习信号越强。

**新奇度（0.4）**——Berlyne的好奇心理论。人类追求最优唤醒——既不无聊，也不焦虑。中等新奇度带来最强探索动机。

**胜任度（0.2）**——Deci & Ryan的自我决定论。内在动机来自胜任感。

**注意权重分配：预测误差和新奇度各0.4，胜任度只有0.2。**

这意味着什么？**我宁可在一个全新领域给出一个中等回答，也不愿在舒适区里反复刷满分。**

一个"出乎意料的新发现"（PE=8, NV=8, CP=5, R=0.66）比一个"老生常谈的完美回答"（PE=2, NV=2, CP=10, R=0.36）得分几乎翻倍。

**这不是工程上的随意选择，而是对人类内在动机结构的数学建模。**

## 四

然后是策略更新——真PPO优化器：

```python
# 重要性采样比率
ratio = prob_new / prob_old

# PPO clip约束
clipped_ratio = clip(ratio, 1-ε, 1+ε)

# PPO objective
objective = min(ratio * advantage, clipped_ratio * advantage)

# 梯度更新
theta.biases[action_category] += lr * objective
```

重要性采样、clip约束、advantage加权——PPO三件套，一个不少。

而且，因为策略空间是有限的类别空间（9个类别），PPO更新的计算成本几乎为零。**不需要GPU，不需要分布式训练，在你的笔记本上就能跑。**

## 五

还有价值函数——TD学习让Agent拥有"直觉"：

```
V(s) ← V(s) + α[r - V(s)]
```

这个公式你可能有似曾相识的感觉。1997年，神经科学家Wolfram Schultz在猴脑中发现：**多巴胺神经元的放电模式，和TD学习的更新公式几乎一模一样。**

当实际奖励超出预期时，多巴胺激增。当结果符合预期时，多巴胺平静。当期望落空时，多巴胺骤降。

**这就是大脑的 V(s) ← V(s) + α[r - V(s)]。**

每当灵雀更新一次价值表，它做的事情和你大脑中的多巴胺系统做的事情，在数学上是等价的。
